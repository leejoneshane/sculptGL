## SculptGL - WebGL sculpting

## Tools
Nodejs needs to be installed nodejs.

Then for the browser build :

yarn # npm install
yarn dev # npm run dev
// visit app/index.html
For standalone : yarn add electron yarn add electron-packager yarn standlone
